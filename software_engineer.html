Resume Management System with AI Parsing

Built a full-stack resume management platform using React, Node.js, Express, and PostgreSQL. Integrated Llama3-based AI model and a custom spaCy NER model to automatically extract structured fields (education, work experience) from uploaded resumes, replacing traditional rule-based parsing. Implemented resume upload, user confirmation UI, field editing, HR management interface, and secure JWT authentication.

Tech Stack: React, Node.js, Express, PostgreSQL, Llama3, spaCy, Ollama, Local LLM, JWT

Built a full-stack e-commerce platform using Ruby on Rails (API-only backend), PostgreSQL, and JWT for secure user authentication and authorization. Added CORS support to enable frontend-backend communication across domains.

Tech Stack: Ruby on Rails (API-only), React, PostgreSQL, Tailwind CSS, Supabase, Netlify, Render, JWT
